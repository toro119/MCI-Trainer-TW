MCI Trainer TW 多人連線核心版 v1.0

目前可測試角色：
- 教官
- 指揮官
- 一次檢傷官
- 二次檢傷官
- 後送官
- 戰情中心

目前功能：
- 教官建立6位數房間碼
- 多支手機加入同一場演練
- 傷患與票色即時同步
- 指揮官隨機派遣91／92救護車並立即到場
- 二次檢傷結果更新戰情資料
- 後送官依載送規則安排1至3名傷患
- 救護車送醫後不可再次使用
- 救護車不足時可通知指揮官
- 戰情中心同步顯示傷患、票色與後送狀態

電腦啟動方式：
1. 安裝 Node.js 18以上版本。
2. 在專案資料夾開啟終端機。
3. 執行 npm install
4. 執行 npm start
5. 電腦瀏覽器開啟 http://localhost:3000

手機加入方式：
1. 手機與電腦連接同一個Wi-Fi。
2. 查詢電腦區域網路IP，例如192.168.1.100。
3. 手機瀏覽器開啟 http://192.168.1.100:3000
4. 教官建立房間，其他手機輸入6位數房間碼加入。

目前限制：
- 伺服器關閉後房間資料會消失。
- 尚未加入正式資料庫、QR Code、治療官及雲端網址。
- 不同網路連線需要後續部署到公開伺服器。
